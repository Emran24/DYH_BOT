# РЕМОНТ.APP — CustDev Bot (PostgreSQL)

Телеграм-бот для глубинных интервью с людьми, которые недавно сделали ремонт.
Деплой на Railway с PostgreSQL — данные не теряются никогда.

---

## Деплой на Railway (шаг за шагом)

### Шаг 1 — Получи токен бота
Напиши **@BotFather** → `/newbot` → скопируй токен вида `1234567890:AAxxx...`

### Шаг 2 — Узнай свой Telegram ID
Напиши **@userinfobot** → скопируй числовой ID, например `123456789`

### Шаг 3 — Залей код на GitHub

```bash
git init
git add .
git commit -m "init custdev bot"
git remote add origin https://github.com/ТВО_ИМЯ/remont-bot.git
git push -u origin main
```

### Шаг 4 — Создай проект на Railway

1. Зайди на [railway.app](https://railway.app) → **New Project**
2. Выбери **Deploy from GitHub repo** → выбери `remont-bot`
3. Railway начнёт сборку — подожди ~1 минуту

### Шаг 5 — Добавь PostgreSQL

1. В проекте нажми **+ New** → **Database** → **Add PostgreSQL**
2. Railway создаст базу и автоматически добавит `DATABASE_URL` в переменные окружения
3. Больше ничего делать не нужно — бот сам создаст таблицу при первом запуске

### Шаг 6 — Добавь переменные окружения бота

Railway → твой сервис (не база) → вкладка **Variables** → добавь:

| Key       | Value                     |
|-----------|---------------------------|
| BOT_TOKEN | токен от @BotFather       |
| ADMIN_IDS | твой числовой Telegram ID |

После добавления Railway автоматически перезапустит бота.

### Шаг 7 — Проверь

- Вкладка **Deployments** → статус `Active`
- Напиши своему боту `/start` — он должен ответить

## Команды администратора

| Команда  | Что делает                          |
|----------|-------------------------------------|
| /start   | Начать интервью                     |
| /cancel  | Прервать интервью                   |
| /stats   | Статистика: начали / завершили / %  |
| /export  | Скачать CSV со всеми ответами       |
